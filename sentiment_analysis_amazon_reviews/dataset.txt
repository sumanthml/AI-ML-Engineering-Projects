📌 Title:
Customer Segmentation Dataset (Mall Customers)

📊 Description:
This dataset contains basic details about mall customers, including:
- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1–100)

It is commonly used for clustering and segmentation projects in Machine Learning.
